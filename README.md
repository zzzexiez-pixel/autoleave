# AutoLeave (Fabric 1.21.x, Loader 0.16.11, JDK 21)

Мод автоматически отключает вас от сервера, если HP падает ниже указанного порога.

## Возможности
- Автовыход при низком HP.
- Меню настроек на клавишу **P**.
- В меню можно:
  - включить/выключить мод;
  - указать HP-порог (от 1 до 20).

## Конфиг
Конфигурация хранится в `config/autoleave.json`.

## Сборка
Требования:
- Gradle **9.3.1+** (только ветка 9.x)
- JDK **21**

Команда сборки:
- Windows/Linux/macOS: `gradle build`

> Проект обновлён под Gradle 9.x и Fabric Loom `1.10.4`.
